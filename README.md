# Resume-builder
A web application that helps users build professional resumes and export them as PDF.

## 🚀Live Demo
[View Live Demo](https://vks-5.github.io/Resume-builder/)

## ⚡Features
- Create professional resumes
- Live resume preview 
- Download as PDF
- Responsive UI and Hover effects
- Contact icons

## ⚙️Technologies used
- HTML
- CSS
- Javascript

## 🟩Libraries used
- Font Awesome
- html2pdf.js

## 🗃️Project Structure
```text
resume-builder/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 📑How to use
1. Open the website
2. Enter resume details
3. click **Generate Resume** button
4. Preview the resume
5. click **Download PDF** button in up to download pdf
